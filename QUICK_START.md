# 🚀 Quick Start - 30 Seconds to Demo

## Start Servers

```bash
# Terminal 1 - Frontend
cd "ai app/frontend"
npm run dev

# Terminal 2 - Backend  
cd "ai app/backend"
npm run dev
```

## Access App

- **Frontend**: http://localhost:3000
- **Backend**: http://localhost:5001

## Login

- **Phone**: `9182146476`
- **OTP**: Any 6 digits

## Demo Flow

1. Login → Home
2. Click "Create New Video"
3. Fill form (any data) → Submit
4. Auto-redirects through:
   - Request Submitted (5s)
   - Call Confirmation
   - Payment
   - Processing (~30s)
   - Video Delivery
