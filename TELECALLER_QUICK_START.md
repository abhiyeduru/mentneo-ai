# ☎️ Telecaller Portal - Quick Start

## 🚀 30 Seconds to Telecaller Demo

### Access Telecaller Portal
- **URL**: http://localhost:3000/telecaller-login
- **Email**: `telecaller@demo.com`
- **Password**: `demo123`

### Demo Flow (2 minutes)

1. **Login** → See dashboard with 4 incoming requests
2. **Accept Call** → Opens call modal
3. **Add Notes** → Type feedback about the call
4. **Select Plan** → Choose Standard (recommended)
5. **Complete Call** → Goes to feedback form
6. **Rate Call** → Give 5-star rating
7. **Add Feedback** → Detailed comments
8. **Check Items** → Verify call checklist
9. **Select Status** → Mark as "Confirmed"
10. **Save** → Success screen
11. **View History** → See all calls & earnings
